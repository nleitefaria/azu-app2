# azu-app2

This application is a RESTful API implementation for an H2 Database
<br /><br /><br />


# To run locally

1. clone or download the project
2. mvn clean install
3. mvn spring-boot:run
4. [http://localhost:8080/](http://localhost:8080/)
<br /><br /><br />
